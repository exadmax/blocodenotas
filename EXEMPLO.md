# Arquivo de Exemplo - Bloco de Notas

Este é um arquivo de exemplo que demonstra as capacidades do aplicativo.

## Funcionalidades de Texto

Este aplicativo suporta **edição de texto** completa, incluindo:

- Criação de arquivos novos
- Salvamento de arquivos
- Carregamento de arquivos salvos
- Exclusão de arquivos

### Edição

Você pode usar todas as funcionalidades de edição:

1. **Recortar**: Remove o texto selecionado
2. **Copiar**: Copia o texto selecionado
3. **Selecionar Tudo**: Seleciona todo o conteúdo

## Tipos de Arquivo

### Arquivos .txt
Arquivos de texto simples para notas rápidas e documentos básicos.

### Arquivos .md
Arquivos Markdown com suporte a formatação rica, como:

- Títulos (H1, H2, H3, etc.)
- **Negrito** e *itálico*
- Listas ordenadas e não ordenadas
- `Código inline`
- Blocos de código
- Links e imagens
- Citações

## Modos de Visualização

### Modo Normal
- Permite edição completa
- Mostra o texto puro
- Ideal para escrever e modificar

### Modo Formato Rich
- Renderiza Markdown
- Modo de visualização
- Mostra o resultado final
